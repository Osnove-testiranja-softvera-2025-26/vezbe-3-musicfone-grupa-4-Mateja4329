MusicFone Testing

Explanation: MusicFone is a music mobile app which supports mobile devices that can convert MP3 and support GPS. The app gives recommended songs and authors to the user which he likes to listen to by using his preferences data. The app can find upcoming songs or events which are connected to those authors using "Last.Fm" website. Functions allow the app to generate recommendations and travel planning for the given events of those authors.

3. If it's successful, then check if we can enter the application.

TEST 2:

TEST 3:
